Installation
============

To install :code:`nessai-gw`, run:

.. code-block:: bash

    pip install nessai-gw

this will also install :code:`nessai` if it is not already installed.

.. important::

    :code:`nessai-gw` requires :code:`nessai>=0.14.0`.

.. note::

    :code:`conda` will be supported in the future.
